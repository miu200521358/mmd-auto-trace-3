■　同梱ファイル
----------------------------------------------------------------

　・readme.txt 　　　　　　　　　 … このファイル
　・alphapose.mp4　　　　　　　　 … AlphaPose解析結果
　・01_json　　　　　　　　　　　 … トレース結果JSONフォルダ
　　　・xxxx.json　　　　　　　　 … JSONファイル
　　　・viz_pos.html　　　　　　　… トレース結果描画HTML
　・02_motion　　　　　　　　　　 … トレース結果モーションフォルダ
　　　・trace_noXXX.pmx　　　　　 … トレース元モデルと同じ骨格の棒人間モデル
　　　・trace_noXXX_original.vmd　… 全打ちモーション
　　　・trace_noXXX_thining.vmd　 … 間引き済みモーション
　・trace_model/trace_model.pmx　 … トレース元モデル（メッシュ入り）

　・jsonやvmdの番号は、AlphaPose解析結果と合わせてあります

■　トレース結果JSONデータの内容
----------------------------------------------------------------

　・単位　　　…　地面からセンチメートル
　　　　　　　　　（ジャンプなどの地面からの距離は大体です）
　・簡易描画　…　viz_pos.html の 冒頭箇所に json データをドラッグ＆ドロップすると、JSONデータの3D描画を閲覧することができます。
